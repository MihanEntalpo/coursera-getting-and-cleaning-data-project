Эксперименты были проведены над группой из 30 волонтёров в возрасте от 19 до 48 лет. Каждый испытуемый выполнять шесть различных видов действий (хотьба. подъём по лестнице, спуск с лестницы, сидение, стояние, лежание), имея при себе смартфон (Samsung Galaxy S II) на поясе. При помощи его встроенного акселерометра и гироскопа мы захватили 3-осное линейное ускорение и 3-х-осные угловые скорости с постоянной частотой в 50 Гц. Эксперименты были записаны на видео, чтобы вручную правильно сопоставить всем данным названия. Полученный набор данных было случайно разделён на два множества, где 70% волонтёров были выбраны в качестве источников данных для генерирования тренировочных данных (для обучения системы), а остальные 30% - для тестирования результатов обучения системы.

Сигналы от сенсоров (акселерометра и гироскопа) были предварительно обработаны путём применения шумоподавляющих фильтров, после чего разбиты по скользящим окнам длинной в 2.56 секунды и с 50% нахлёстом (128 чтений на окно). Сигналы акселерометра, которые включают в себя гравитационную компоненту и компоненту движения тела, были разделены с помощью фильтра Баттерворта на "гравитацию" и "ускорения тела". Предпологается, что сила тяжести представлена только низкочастотными компонентами, поэтому для фильтрации была использована частота 0.3 Гц. Из каждого окна был получен вектор значений, путём вычисления переменных из времени и частоты. Смотрите "features_info.txt", чтобы узнать подробности.

Для каждой записи предоставляется:
======================================

- Трёхосное ускорение полученное от акселерометра (суммарное) и приблизительное ускорение тела
- Трёхосная угловая скорость от гироскопа
- Вектора времени и частоты по 561 свойствам
- Название активности 
- Идентификатор субъекта, на котором проводился эксперимент

Набор данных содержит следующие файлы
=========================================

- 'README.txt'

- 'features_info.txt': Содержит информацию о переменных, использованных в векторе свойств

- 'features.txt': Содержит все свойства

- 'activity_labels.txt': Свзываем названия классов с названиями физической активности

- 'train/X_train.txt': Набор данных, используемый для обучения системы.

- 'train/y_train.txt': Подписи данных, используемые для обучения системы.

- 'test/X_test.txt': Набор данных, используемый для проверки обученной системы.

- 'test/y_test.txt': Подписи данных, используемые для проверки обученной системы.

Для тренировочных и тестовых данных доступны следующие файлы. Их описания эквивалентны.

- 'train/subject_train.txt': Each row identifies the subject who performed the activity for each window sample. Its range is from 1 to 30. 

- 'train/Inertial Signals/total_acc_x_train.txt': Данные акселерометра, ось X в единицах "g". Каждая строка - это вектор из 128 элементов. Тоже самое относится к файлам 'total_acc_y_train.txt' и 'total_acc_z_train.txt' для осей Y и Z. 

- 'train/Inertial Signals/body_acc_x_train.txt': Данные акселерометра, ось X, ускорение тела, без силы тяжести

- 'train/Inertial Signals/body_gyro_x_train.txt': Вектор угловых скоростей вокруг оси X, измеренный гироскопом для каждого сэмпла окна. Единицы измерения 0 радианы/секунду.

Заметки:
========
- Переменные нормализованы и приведены к границам [-1, 1]
- Вектор каждой из переменных - это строка в текстовом файле
